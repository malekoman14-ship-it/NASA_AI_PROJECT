Project Description: 
AI-Powered Autonomous Fault Detection and Recovery System for NASA spacecraft. 
This conceptual design proposes a four-module AI architecture using anomaly detection, fault 
classification, and autonomous recovery planning to enable spacecraft to self-diagnose and self-repair 
during deep-space missions without real-time human intervention.

How to Read Documentation:

Project_Proposal.docx, then Solution_Plan.docx, then Testing_Plan.docx, and finally view 
Presentation_Slides.pdf for the visual summary.
